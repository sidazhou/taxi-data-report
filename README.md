To see the results
---------
- see matlab notebook in `./html/task1.html` and `./html/task2.html`
OR
- https://rawgit.com/sidazhou/taxi-data-report/master/html/task1.html
- https://rawgit.com/sidazhou/taxi-data-report/master/html/task2.html
